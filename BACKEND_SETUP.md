# SEDERG PAY - Backend (Yii2)\n\n## 🛠️ Setup\n\n### Requisitos\n- **PHP** 8.1+\n- **MySQL** 8.0+\n- **Composer** 2.0+\n- **XAMPP** (opcional, para desarrollo local)\n\n### Instalación\n\n```bash\ncd backend\ncomposer install\n```\n\n### Configuración\n\n#### 1. Variables de Ambiente\n```bash\ncp .env.example .env\n```\n\nEditar `.env`:\n```env\nDB_HOST=127.0.0.1\nDB_NAME=sederg_pay\nDB_USER=root\nDB_PASS=\nJWT_SECRET=your-secret-key-here\n```\n\n#### 2. Base de Datos\n\n```bash\n# Crear base de datos\nmysql -u root -e \"CREATE DATABASE sederg_pay CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;\"\n\n# Ejecutar migraciones\nphp yii migrate\n```\n\n#### 3. Permisos de Carpetas\n\n```bash\n# Windows no requiere permisos especiales\n\n# macOS/Linux\nchmod -R 755 runtime\nchmod -R 755 web/assets\n```\n\n---\n\n## 🚀 Ejecución\n\n### Desarrollo\n```bash\n# Desde la carpeta backend\nphp -S localhost:8000\n```\n\nDisponible en: **http://localhost:8000**\n\n### Producción\n```bash\n# Apache\nsudo chown -R www-data:www-data /path/to/backend\nsudo chmod -R 755 /path/to/backend\n\n# Configurar Virtual Host\n```\n\n---\n\n## 📂 Estructura\n\n```\nbackend/\n├── config/\n│   ├── db.php            # Database config\n│   ├── web.php           # Web app config\n│   └── params.php        # App params\n├── controllers/\n│   ├── AuthController.php\n│   ├── RemesaController.php\n│   └── BancoController.php\n├── models/\n│   ├── User.php\n│   ├── Remesa.php\n│   ├── Banco.php\n│   └── ...\n├── migrations/\n│   └── m*.php            # DB migrations\n├── views/\n├── web/\n│   └── index.php         # Entry point\n├── runtime/\n├── vendor/               # Composer dependencies\n└── yii                   # CLI tool\n```\n\n---\n\n## 🔐 Autenticación JWT\n\nEl backend usa **JWT (JSON Web Tokens)** para autenticación stateless.\n\n### Headers Requeridos\n```\nAuthorization: Bearer <token>\nContent-Type: application/json\n```\n\n### Token Claims\n```json\n{\n  \"userId\": 1,\n  \"email\": \"user@example.com\",\n  \"iat\": 1234567890,\n  \"exp\": 1234654290\n}\n```\n\n---\n\n## 📊 Base de Datos\n\n### Tablas Principales\n- `user` - Usuarios registrados\n- `remesa` - Transacciones de remesas\n- `banco` - Instituciones bancarias\n- `pais` - Países soportados\n- `tasa_cambio` - Tipos de cambio actuales\n\n### Diagrama ER\n```\nuser (1) ──→ (N) remesa\n        └──→ (N) referencia_bancaria\nbanco (1) ──→ (N) remesa\npais (1)  ──→ (N) user\n        └──→ (N) banco\n```\n\n---\n\n## 🧪 Testing\n\n```bash\n# Listar todos los endpoints\nphp yii help\n\n# Test de conexión a BD\nphp yii migrate/status\n\n# Crear usuario test\nphp yii db/seed\n```\n\n---\n\n## 📝 API Endpoints\n\nVer [API_REFERENCE.md](../API_REFERENCE.md) para documentación completa.\n\n### Resumen\n- `POST /auth/login` - Login\n- `POST /auth/registro` - Registro\n- `GET /auth/perfil` - Perfil del usuario\n- `GET /remesa` - Listar remesas\n- `POST /remesa/crear` - Crear remesa\n- `GET /banco` - Listar bancos\n\n---\n\n## 🐛 Troubleshooting\n\n### Error: \"Database connection failed\"\n- Verificar credenciales en `.env`\n- Asegurar que MySQL está corriendo\n- Verificar que la BD existe\n\n### Error 401 Unauthorized\n- Token JWT inválido o expirado\n- Header `Authorization` no enviado\n- Token no coincide con JWT_SECRET\n\n### Error: \"Table already exists\"\n```bash\nphp yii migrate/down --limit=1\nphp yii migrate\n```\n\n---\n\n## 📚 Recursos\n\n- [Yii2 Guide](https://www.yiiframework.com/doc/guide/2.0)\n- [Yii2 API](https://www.yiiframework.com/doc/api/2.0)\n- [JWT.io](https://jwt.io)\n- [RESTful API Design](https://restfulapi.net)\n\n---\n\n## ✅ Checklist de Deployment\n\n- [ ] Variables de ambiente configuradas\n- [ ] BD migrada y poblada\n- [ ] JWT_SECRET establecido\n- [ ] CORS configurado\n- [ ] SSL/HTTPS habilitado\n- [ ] Rate limiting activo\n- [ ] Logs configurados\n- [ ] Backups programados\n\n---\n\nMás info: Ver [README.md](../README.md) principal\n
