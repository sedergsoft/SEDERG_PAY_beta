# 🚀 Guía de Despliegue a Producción - SEDERG PAY\n\n## 📋 Checklist Pre-Despliegue\n\n### Frontend\n- [ ] Configurar variables de ambiente (.env)\n- [ ] Verificar API_URL apunta a backend en producción\n- [ ] Ejecutar `npm run build`\n- [ ] Verificar bundle size (~150KB)\n- [ ] Probar build localmente\n\n### Backend\n- [ ] Configurar variables de ambiente (.env)\n- [ ] Cambiar JWT_SECRET a valor fuerte\n- [ ] Configurar BD de producción\n- [ ] Ejecutar migraciones\n- [ ] Verificar conexión a BD\n\n### Base de Datos\n- [ ] Backup de datos actuales\n- [ ] BD creada en servidor\n- [ ] Credenciales seguras\n- [ ] Replicación automática\n\n---\n\n## 🏢 Despliegue Rápido - XAMPP/Servidor Local\n\n### Frontend\n```bash\ncd frontend\nnpm install\nnpm run build\nnpx serve -s dist -l 3000\n```\n\n### Backend\n```bash\ncd backend\ncp .env.example .env\n# Editar .env\ncomposer install\nphp yii migrate\nphp -S 0.0.0.0:8000\n```\n\n---\n\n## 🐳 Despliegue con Docker\n\n### docker-compose.yml\n```yaml\nversion: '3.8'\nservices:\n  db:\n    image: mysql:8.0\n    environment:\n      MYSQL_DATABASE: sederg_pay\n      MYSQL_ROOT_PASSWORD: secure-password\n    ports:\n      - \"3306:3306\"\n\n  backend:\n    build: ./backend\n    ports:\n      - \"8000:80\"\n    environment:\n      DB_HOST: db\n      DB_NAME: sederg_pay\n      JWT_SECRET: your-secret-key\n    depends_on:\n      - db\n\n  frontend:\n    build: ./frontend\n    ports:\n      - \"3000:3000\"\n```\n\n### Ejecutar\n```bash\ndocker-compose up -d\n```\n\n---\n\n## ☁️ Despliegue en VPS\n\n### Requisitos\n- PHP 8.1+\n- MySQL 8.0+\n- Node.js 18+\n- Nginx/Apache\n- SSL/TLS\n\n### Instalación\n```bash\n# Clonar repositorio\ncd /var/www\ngit clone https://github.com/sedergsoft/SEDERG_PAY_beta.git\ncd SEDERG_PAY_beta\n\n# Configurar Backend\ncd backend\ncomposer install --no-dev\nphp yii migrate\n\n# Configurar Frontend\ncd ../frontend\nnpm install\nnpm run build\n```\n\n### Nginx Configuration\n```nginx\nserver {\n    listen 80;\n    server_name your-domain.com;\n    \n    location / {\n        root /var/www/SEDERG_PAY_beta/frontend/dist;\n        try_files $uri /index.html;\n    }\n    \n    location /api {\n        proxy_pass http://127.0.0.1:8000;\n    }\n}\n```\n\n---\n\n**Última actualización**: 2026-05-27\n
